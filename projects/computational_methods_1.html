Adaptive Mesh and Basis Function Analysis for Nonlinear Diffusion-Reaction Equations

View Repository

Overview

Conducted an in-depth numerical investigation of dimensionless nonlinear diffusion-reaction boundary value problems using the Galerkin Finite Element Method (GFEM). The project specifically explored the use of linear and quadratic Lagrange basis functions integrated with a three-point Gaussian Quadrature within a computational domain of unit length.

Key Contributions

  • Boundary Value Problem Setup: Addressed the nonlinear-chemical reaction equation:
    $$ \frac{d^{2}C}{dx^{2}} - PC^{3} = 0, $$ with boundary conditions $$ \left.\frac{dC}{dx}\right|_{x=0} = 0 $$ and $$C(1) = 1.$$
  • Mesh Independence Study: Evaluated how node quantity influences solution accuracy and indetified conditions for achieving mesh independence. Demonstrated that increasing node reduces error significantly up to a thresold after which additional nodes have minimal impact.
  • Mesh Distribution Optimization: Explored effects of non-uniform mesh distributions, identifying strategies for densely packing nodes in high-gradient regions(steep slopes or boundary layers) and sparsely packing nodes in low-gradient regions. Determined that careful node placement significantly enhances solution precision.
  • Basis Function Comparative Analysis: Conducted comparative experiments between linear and quadratic basis functions, concluding quadratic basis functions require fewer nodes for similar accuracy. Highlighted the necessity of function behavior-based node spacing rather than arbitrary selection.

Outcomes

  • Established qualitative insights into selecting optimal mesh and basis function strategies.
  • Defined clear guidelines for balancing computational efficiency and solution accuracy in solving nonlinear diffusion-reaction equations.
