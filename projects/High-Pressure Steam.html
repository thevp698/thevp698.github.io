Overview

Implemented an advanced optimization tool in a real-world industrial scenario to optimize the compression process, elevating incoming steam pressure effectively.

Key Features

  • Mathematical Modeling: Developed a robust mathematical model addressing complex nonlinear constrained optimization problems. The model precisely incorporated critical system constraints, such as mass and energy balances, dew point conditions, and allocation fractions.
  • Blackbox Optimization: Employed advanced blackbox optimization techniques, leveraging the open-source NOMAD optimizer developed by Polytechnique Montreal.
  • Performance Enhancements: Optimized existing system designs, achieving substantial improvements and surpassing previous key performance indicators (KPIs) by 10%.
  • Custom Python API

Benefits

  • Increased efficiency and reliability in steam compression processes.
  • Provided optimized, feasible solutions to complex industrial constraints.
  • Demonstrated significant measurable improvements in operational performance.

Technologies Used

This project utilized Aspen HYSYS as a simulation tool to model the process systems. In addition, we experimented with various optimization tools to solve process optimization problems.

Tools

  • Git & GitHub for version control
  • PyTest for testing
  • Aspen HYSYS for simulation
  • Win32 for automation
  • NOMAD, Simcenter Studio & HEEDS for optimization
